# AuditAI Architecture

## System Overview

AuditAI is an AI-powered infrastructure auditor for Google Cloud Platform.

### Components

1. **Backend** - FastAPI application
   - GCP integration
   - Gemini AI analysis
   - MongoDB persistence

2. **Frontend** - Next.js dashboard
   - Real-time metrics
   - Authentication
   - Recommendations display

3. **Orchestration** - Kestra workflows
   - Scheduled audits
   - Alert notifications
   - Report generation

### Tech Stack

- Backend: Python, FastAPI
- Frontend: Next.js, React, Tailwind
- AI: Google Gemini API
- Database: MongoDB
- Orchestration: Kestra
- Deployment: Vercel
